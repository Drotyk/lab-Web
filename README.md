Статичний вебпроєкт про гурт **M8L8TH**. Сайт містить кілька тематичних сторінок, каталог альбомів, окремі сторінки релізів і просту мінігру на JavaScript.

## Опис

Проєкт реалізований без фреймворків і збірки. Уся структура побудована на звичайних `HTML`, `CSS` і `JavaScript`, тому сайт можна швидко запускати локально, редагувати вручну та розгортати на будь-якому статичному хостингу.

## Функціональність

- головна сторінка з навігацією та візуальними елементами;
- сторінка `Про нас`;
- сторінка з контактною інформацією;
- сторінка з розкладом туру;
- каталог альбомів;
- окремі сторінки для кожного альбому;
- мінігра-клікер з модальним вікном;
- прості інтерактивні елементи на JavaScript.

## Технології

- `HTML5`

Зовнішні бібліотеки, пакетні менеджери та етап збірки не використовуються.

## Локальний запуск

### Варіант 1: напряму в браузері

Відкрийте файл `index.html`.

### Варіант 2: локальний сервер

```bash
python3 -m http.server 8000
```

Після запуску відкрийте:

```text
http://localhost:8000
```

## Основні сторінки

- `index.html` — головна сторінка;
- `about.html` — коротка інформація про сайт;
- `tour.html` — сторінка з розкладом туру;
- `contacts.html` — контакти;
- `game.html` — клікер-гра;
- `albums.html` — каталог альбомів;
- `albums/*.html` — окремі сторінки релізів.

## JavaScript

- `assets/js/script.js` — логіка перетягування логотипа на головній сторінці;
- `assets/js/game.js` — механіка клікер-гри, рахунок і модальне вікно.

## Деплой на GitHub Pages

Оскільки це статичний сайт, його можна розгорнути через GitHub Pages без додаткової збірки.

1. Відкрити репозиторій на GitHub.
2. Перейти в `Settings`.
3. Відкрити розділ `Pages`.
4. У полі `Source` вибрати `Deploy from a branch`.
5. Вибрати гілку `main`.
6. Вибрати папку `/ (root)`.
7. Зберегти налаштування.

Після цього GitHub Pages згенерує публічне посилання на сайт.
